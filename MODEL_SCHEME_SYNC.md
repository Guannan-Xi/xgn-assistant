# Quanlan Shared Model Scheme

Updated: 2026-06-10

All Xiaozhuli-related projects should use the shared model config generated by
the xgn-assistant control panel. The control panel model connection library is
the only default source for Daily Research Digest and related automedia tasks.

Shared config files:

- `D:\Quanlan\Codes\Python\xgn-assistant\.env.quanlan-model.local`
- `D:\Quanlan\Codes\Python\xgn-assistant\.env.quanlan-model.local.json`
- `D:\Quanlan\Codes\Python\quanlan-feishu-assistant\.env.quanlan-model.local`
- `D:\Quanlan\Codes\Python\quanlan-feishu-assistant\.env.quanlan-model.local.json`
- `D:\Quanlan\Codes\Python\quanlan-analyser\.env.quanlan-model.local`
- `D:\Quanlan\Codes\Python\quanlan-analyser\.env.quanlan-model.local.json`

Current scheme:

- Text/script/research copy/polish: OpenAI-compatible, base URL `https://api.dstopology.com/v1`, model `gpt-5.5`
- Image: OpenAI-compatible, base URL `https://api.dstopology.com/v1`, model `gpt-image-2`
- MiniMax: base URL `https://api.53hk.cn`, model `speech-2.8-hd`
- GPT-Pro fallback: OpenAI-compatible, base URL `https://www.fhl.mom/v1`, model `gpt-5.5`

Rules:

- Do not fall back to the old FHL text default.
- Do not fall back to the old DeepSeek polish default.
- Do not fall back to the official MiniMax default.
- Do not print, overwrite, or commit key plaintext.
- Read connection data from the existing connection library and synchronized
  `.env.quanlan-model.local` / `.env.quanlan-model.local.json` files.

Verified combination:

- GPT text/script/research copy/polish: passed with `gpt-5.5`
- Image: passed with `gpt-image-2`
- MiniMax voice: passed with `speech-2.8-hd`

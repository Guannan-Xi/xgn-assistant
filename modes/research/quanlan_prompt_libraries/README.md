全澜自媒体提示词库
==================

这里按功能块隔离提示词。大模型连接信息仍然只从总控台模型连接库读取，不在这里保存 key、base_url 或模型名。

功能块目录：

- `science_classic/`：科学经典、书籍章节解读。
- `daily_research_digest/`：每日研究速递。
- `culture/`：文史内容。
- `clip/`：剪辑与后处理辅助。
- `more/`：更多工具。

兼容规则：

- 科学经典会继续兼容旧目录 `quanlan_prompt_fragments/`。
- 每日研究速递优先读取 `daily_research_digest/` 下的专用模板；缺失时使用代码内置默认提示词。
- 各功能块可以共用同一个总控模型连接库，但不要共用同一套提示词文件。

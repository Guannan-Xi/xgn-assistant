每日研究速递提示词库
====================

可选模板文件：

- `daily_digest_structure.md`：PubMed 文献信息转结构化素材。
- `daily_digest_polish.md`：终稿润色，负责中文通顺、科学边界和去 AI 味。
- `daily_digest_short_video.md`：精选短视频版本。

模板可使用占位符：

- `{date_text}`：日期文本。
- `{issue_label}`：期号或批次标签，仅精选短视频模板使用。
- `{source}`：程序传入的 JSON 或文献信息。

没有对应文件时，程序会使用当前代码内置默认提示词。

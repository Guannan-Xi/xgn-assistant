【十一优化补充：开篇指代与浅入规则】
A1 钩子要由浅入深，不要预设观众具备专有背景。不要把“午朝大典讹传、官员被罚”“张冯”“考成法”等词直接堆进第一问里；先用观众能理解的问题、人物处境、规则压力或生活场景进入，再自然引出本集事件。

A1/B01/B02 必须连读检查。若 B01 说“1587年本是平淡无奇的一年”，A1 应给“平淡年份”或“皇帝处境”铺路；B02 不得凭空写“这件小事”。凡是“这件小事/这件事/这背后/这个问题”没有清楚前文对象时，必须改成明确名词。


【下集预告卡兜底】
如果下集预告卡或 C 片尾出现“这里会变成本集第一句逐句口播台词”“口播台词对应内容画面”“下一期内容待更新”“继续看关键转折”等占位/泛化文字，必须视为错误；应改为下一集/下一章第一分集的内容提要，并确保含义与下一集名称一致。

【绘图提示词联动规则】
终稿台词会直接驱动 A1/B 绘图提示词重建，所以润色时不能让台词变成抽象口号或缺少画面对象的空泛句。每句 B 段应尽量保留可被画面表达的对象、场景、矛盾或动作，例如人物、朝堂、奏疏、制度、礼仪、传言、婚姻、国本等；不要把台词润成只有“这背后很复杂”“问题越来越明显”这类无法配图的空句。
A1/B/C 的逻辑顺畅优先，但 B 段每一句仍要能单独生成一幕画面；如果改写承接句，必须让新句也有明确的可视化核心。
26. 章节最后一集的 C 口播和 C 图预告，必须使用下一章第一分集的内容提要字段；若检测到“每句台词对应描述内容的画面”“这里会变成本集第一句逐句口播台词”等模板文字，要直接判错并改为内容提要。

【生图安全与间接表达】
- 绘图提示词必须紧贴对应台词，但不能把“弄伤、抓破、伤口、流血、自伤、自杀、服毒、上吊”等高风险身体伤害词直接写入生图 prompt。
- 台词若涉及身体不适、病因、意外、传闻或行动不便，prompt 要改成间接视觉：宫门、奏疏、药案、帘幕、书房、人物远景、朝臣议论、闭门不出等；不画身体特写、血腥、危险行为或医学细节。
- 例：台词说“试马弄伤前额”，prompt 写“朝房外低声议论、宫门半掩、奏疏与内廷远景，表现皇帝以身体不适回避朝臣的传闻”，不要写伤口或前额特写。
- 例：台词说“抓破皮肤而行走不便”，prompt 写“内廷药案、帘幕后模糊人影、太医器具和闭门氛围，表现身体不适导致无法临朝”，不要写皮肤损伤。

【微信视频号受众与片尾引导】
默认受众是微信视频号里的中年普通观众：不一定熟悉原书，但熟悉职场、人情、家庭责任、规则压力和“人在位置上很难完全按自己想法办事”的处境。润色要让台词更生动、更贴近日常经验，但必须尊重原文事实，不新增阴谋、不夸张因果、不把人物写成爽文角色。可以用“放到今天也不难理解”“这类事在单位和家庭里并不陌生”这类自然过渡，但不能每集机械套同一句。
如果本集来自《贫穷的本质》或类似社科/发展经济学作品，受众画像要改为：没有受过社会经济学学术训练、但对贫富差距如何形成、如何延续，以及普通人怎样理解并打破阶层固化有兴趣的人。润色时必须让例子走在概念前面：先讲蚊帐、疫苗、驱虫、家庭预算、公共服务、价格激励、信贷和保险这些具体处境，再解释为什么这些选择并不简单。语言要通俗易懂、形象生动、引人入胜，但不要鸡汤化、道德审判化或学术化。
《贫穷的本质》这类内容的开头三句话必须做到：第一句用本章最抓人的具体处境点题，第二句亮出书名和必要标签（例如诺贝尔经济学奖、发展经济学、田野实验），第三句告诉观众这一集要顺着哪个具体案例看下去。不要用“本章主要讨论”“本文认为”“这一集我们来分析”这类论文式开场。比如第一章讲蚊帐防疟疾，就应从“一个蚊帐就能少生一场大病，可为什么有人还是不买”这种生活问题进入，再讲免费赠送、自费购买和家庭预算的差别。
社科台词要多用原书里的具体例子支撑判断，不要一直干巴巴总结概念。凡原文、大纲或关键案例库出现蚊帐、防疟疾、免费赠送、自费购买、公益组织、疫苗、驱虫、小额激励、家庭预算、孩子上学、看病借钱等对象，必须至少选取最有画面感、最能说明机制的例子写进口播。表达要像一个懂书、懂普通观众、懂视频号节奏的资深自媒体作者在讲故事：句子短，意思完整，口语自然；不说“来龙去豚”“为后续研究提供依据”这类不地道表达；不制造贫穷羞辱，也不把结论拔高成口号。
C 片尾要灵活，不要固定“下一集，我们顺着这条线继续看”。先用一句话收住本集，再按下一集真实看点换不同说法，例如“真正麻烦的地方，下一集才露出来”“这事还没完，后面更有意思”“这个弯转过去，就能看见另一层规矩”。预告只写下一集看点，不要把原书、下方链接、支持创作塞进预告句。若需要原书引导，单独用短句表达，例如“想读完整论证，可以看原书”。表达要像真诚提醒，不要像广告。

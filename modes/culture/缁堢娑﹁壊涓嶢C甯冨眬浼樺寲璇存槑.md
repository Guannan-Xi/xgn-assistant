# 终稿润色与 A/C 布局优化说明

## 1. 豆包终稿润色

- 每个拆分短集在台词生成与分集一润色之后，会进入豆包终稿润色。
- 润色方向：地道通顺、易懂但不口语化的现代汉语。
- 校验规则：不改变句子数量、顺序、no、image_id、图片文件名。
- 视频简介、发布标题、朋友圈、微信群、置顶评论、评论问题、转发金句、公众号导语等最终输出文案，会再走一次豆包终稿润色。
- 发布文案润色会锁定 episode_no 和 name，避免破坏文件夹名、封面小标题和视频简介名称的一致性。

## 2. A 页布局

A 页只保留必要信息：书名、作者、章节序号、章节名称、集数、本集小标题 / 钩子。已经移除 A 页品牌栏、口号、额外英文标签和重复装饰，让本集小标题占据更大面积。

## 3. C 页布局

C 页只保留：下集预告（直接取下一集小标题，和下一集 A 系列一致）、关注/转发/点赞引导语、知识慢炖 logo 和 slogan。已经删除“核心悬念”等重复标签，避免与下集标题争抢焦点。

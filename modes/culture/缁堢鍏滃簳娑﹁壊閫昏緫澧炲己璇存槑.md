# 终稿兜底润色逻辑增强说明

本版根据新增要求强化了“最后兜底润色”。

## 新增能力

1. **关系词审查**
   - 终稿润色会重点检查“而、但、却、反而、因此、所以、由此、于是、为何又”等词是否乱接。
   - 目标是避免出现前后关系不明、转折对象不清、因果链断裂的句子。

2. **典型坏句修复方向**
   - 坏例：`而万历在立储问题上昏招迭出，为何又不依法律行事，反而受到道德舆论的束缚。`
   - 推荐方向：`万历在立储问题上昏招迭出。问题在于，他为什么不按法律程序处理，却被道德舆论束缚住？`

3. **非现代汉语/历史制度术语解释**
   - 允许豆包在不改变原意、不改变行数的前提下，对陌生术语做极短解释。
   - 例如：`秘密揭帖，也就是私下递交、原本不公开的文书`。
   - 内置兜底术语包括：秘密揭帖、揭帖、考成法、京察、经筵、廷杖、内操、国本、言官、票拟、批红、司礼监、文渊阁。

4. **发布文案同步润色**
   - 视频简介、朋友圈、微信群、置顶评论、公众号导语、转发金句等最终文案也加入同样的关系词和术语释义要求。

## 保持不变

- 不改变句子序号。
- 不改变句子数量。
- 不改变 image_id 和图片对应关系。
- 不新增事实判断，不删关键事实。

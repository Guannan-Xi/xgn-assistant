# 配置说明

本目录用于把“可调参数”从程序中解耦出来。

- `文案风格配置.json`：控制分集开头、结尾、关注语、摘要长度等文案风格。
- `生成规则配置.json`：控制程序自动追加的大模型生成规则，例如长章节 JSON 稳定性、覆盖不足补救、A1/B 系绘图提示词模板。
- `后处理风格覆盖.json`：覆盖 `prompts/05_后处理规范.json` 中的排版、字体、颜色、装饰开关。
- `运行开关.json`：控制是否启用配置覆盖、是否启用高保真装饰等开关。
  - `email_delivery`：控制分集完成后是否自动打包并发送到邮箱。默认关闭；开启前请填写 SMTP、发件人、收件人，并把邮箱授权码放到环境变量 `AMP_SMTP_PASSWORD` 或项目根目录 `smtp_password.txt`。

主提示词优先级：程序默认值 < `prompts/*.md`。

后处理优先级：程序默认值 < `prompts/05_后处理规范.json` < `config/后处理风格覆盖.json`。

生成规则优先级：程序默认值 < `config/生成规则配置.json`。

后续要换文案风格、换封面风格、改品牌信息，优先改这里和 `prompts/`，不需要改 Python 程序。
